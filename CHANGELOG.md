# 更新日志

本文件用于记录项目的关键版本变更。

## [v1.1.0] - 2026-03-25

### 新增
- 新增任务优先级字段（`high` / `medium` / `low`），支持创建、编辑、列表展示与详情展示。
- 新增任务按优先级排序能力：`GET /tasks` 支持 `sortBy=priority`，`sortOrder` 支持 `high|medium|low`（默认 `high`）。
- 新增项目编码自动生成功能：创建项目时编码可留空，后端自动生成 `PROJ-XXXXXXXX`。
- 新增 API 基准测试脚本与对比脚本：`scripts/benchmark-api.sh`、`scripts/benchmark-compare.sh`。

### 变更
- 完成后端稳健提效改造：请求级权限缓存、写链路事务化、错误响应统一、审计细节模板统一。
- 完成前端数据层统一：收敛分页/数组/对象加载与错误读取，核心页面去除主要 `any`。
- Compose 部署增强：MySQL/Nginx 国内镜像切换，构建镜像与代理参数化，弱网环境可回退拉起。
- Dashboard 图表模块进一步懒加载拆分，降低路由初始负担。

### 修复
- 修复通知接口缺失场景下的降级与轮询行为，避免 404 循环请求。
- 修复通知与权限场景下部分页面提示与状态不一致问题。
- 修复任务/项目相关链路中的部分事务一致性与回滚覆盖不足问题。

## [v1.0.0] - 2026-03-25

### 新增
- 完成基于 `Go (Gin + Gorm + JWT)`、`MySQL`、`Vite + React + TypeScript` 的项目管理系统基础架构。
- 新增 RBAC、用户管理、部门管理、项目管理、任务管理、个人工作台、统计分析、审计日志、甘特图、任务分解树等核心功能页面。
- 新增 `.env.template` 环境变量模板，支持运行时通过环境变量读取配置。
- 新增前端通用能力与组件（`Pagination`、`DataState`、状态常量、UI 规范文档/检查清单）。
- 新增 Docker Compose 与 Helm/K8s 部署相关资源。

### 变更
- 优化项目/任务数据权限范围控制：普通用户按“与我相关”范围查看数据。
- 优化统计分析页面：无用户管理权限时不展示“用户数”卡片。
- 优化项目编辑与任务编辑弹窗，支持搜索选择负责人、部门、执行人。
- 优化甘特图视觉样式，提高进度与状态的可读性和对比度。

### 修复
- 修复本地开发 CORS 跨域问题。
- 修复 RBAC 权限更新后生效不及时的问题。
- 修复多人选择控件必须 `command + 点击` 才能取消的问题（改为复选框交互）。
- 修复项目/任务编辑时用户与部门选项加载不稳定、数据为空等问题。
- 修复统计图与甘特图在空数据场景下的空指针渲染问题。
